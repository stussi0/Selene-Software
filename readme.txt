#Instalacão

1 - Compile o código exec.cpp com o comando:
    g++ exec.cpp -o exec

2 - Execute o shell script build.sh com o comando:
    ./build.sh

3 - Execute o shell script compile.sh com o comando:
    ./compile.sh

Obs: Caso existam os arquivos "dependencies.jar" e "exec" 
     e a pasta "out", apague-os antes da instalacao!

#Rodar o Selene

Para rodar o Projeto Selene basta conectar o arduino no USB 
sem abrir nenhum outro programa e digitar o comando:
    ./exec
